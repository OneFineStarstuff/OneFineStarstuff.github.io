# T4 Incident Notification Playbook Template

## 1. Trigger Criteria
- Sev-1: consumer harm at scale, severe control failure, or systemic risk signal.
- Sev-2: material control degradation with contained scope.

## 2. Initial Triage (0-30 minutes)
- Open incident bridge
- Assign incident commander
- Contain model/workflow if required
- Preserve evidence snapshot

## 3. Notification Matrix
- Internal: CEO/CRO/CISO/GC/Compliance
- External: supervisor/regulator contacts by jurisdiction

## 4. Required Notification Fields
- Incident ID / timestamp (UTC)
- Affected systems and workflows
- Preliminary impact and containment status
- Customer impact summary
- Next update commitment

## 5. Post-Incident
- Root cause analysis
- Corrective actions + owners + due dates
- Validation/retest evidence
